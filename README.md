# Spring Boot Bookstore REST API
A RESTful Online Bookstore API built with Java and Spring Boot, using MongoDB for data storage. The project supports book management, user operations, and order processing.

## Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven
- REST API

## Features (Planned)
- Book management (CRUD)
- User management
- Order management
- Search books by title/author

## Status
Project in development.
